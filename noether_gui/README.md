# Noether GUI

![GUI](../docs/gui.png)

Run the GUI application from its install location in the workspace:
```
cd <ws>/install/noether_gui/bin
./noether_gui_app
```

Alternatively, run the GUI application using the ROS2 framework:

```
ros2 run noether_gui noether_gui_app
```
